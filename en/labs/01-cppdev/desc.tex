\section{Lab description}

In this lab you will get used to the programming environment used in 
several parts of this course:

\begin{itemize}

  \item Operating Sytem: Linux

  \item C++ compiler: \textemph{g++-14} and \textemph{clang++-20}.

  \item IDE: \textemph{VS Code}.

  \item Build manager: \textemph{CMake}.

  \item Dynamic anaysis tools: 
        \textemph{valgrind-memcheck} and 
        compiler \emph{sanitizers}.

\end{itemize}

To carry out this lab you will need to previously intall the \textemph{Docker
Desktop} manager for your operating system.
