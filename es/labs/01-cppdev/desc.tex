\section{Descripción del laboratorio}

En este laboratorio adquirirás experiencia con el entorno de programación que se
utiliza en varias partes de la asignatura.

\begin{itemize}

  \item Sistema operativo: \textemph{Linux}.

  \item Compiladores de C++: \textemph{g++-14} y \textemph{clang++-20}.

  \item Entorno de Desarrollo: \textemph{VS Code}.

  \item Gestor de construcción: \textemph{CMake}.

  \item Herramientas de análisis dinámico: \textemph{valgrind-memcheck} y
\textemph{sanitizers} del compilador.

\end{itemize}

Para llevar a cabo este laboratorio deberás instalar previamente el gestor
\textemph{Docker Desktop} para tu sistema operativo.
